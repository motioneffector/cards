Installation

npm install @motioneffector/cards

Requirements

  • Node.js 18+ or modern browser (ES2022+)
  • TypeScript 5.0+ recommended

Zero Dependencies

This library has no runtime dependencies. PNG parsing, Base64 encoding, and ZIP handling are all implemented internally. No supply chain risk.
